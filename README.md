# Licsber 工具箱

## 代码说明

个人娱乐, 供参考.

## 命令介绍

licsber: Hello world.  
count-dir: 统计目录下的文件与目录数.  
flatten-dir: 递归展开当前目录下所有子文件夹到当前目录.  
memobird: 发送咕咕机消息.
